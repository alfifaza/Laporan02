A. Mengapa Menggunakan CSS dan JavaScript dalam Pemrograman Web?

CSS digunakan untuk mengatur tampilan visual dan gaya elemen halaman web. Seperti mengubah Warna, Font, Tata Letak, Responsivitas (tampilan desktop, tablet, ponsel).

JavaScript digunakan untuk menambahkan interaktivitas dan fungsionalitas halaman web. Seperti mengatur Animasi dan Efek Visual, Reaksi Elemen saat di Klik, meningkatkan UX web, dan menampilkan atau memvalidasi form ketika diisi, dan mengelola fitur web yang canggih.

B. Bagaimana CSS dan JavaScript Dapat Diintegrasikan dalam Pemrograman Web?

CSS (tampilan elemen) pada Internal di dalam tag head, pada Eksternal file terpisah .css dan dihubungkan dengan via link href, dan Inline di atribut style (tidak disarankan).

JavaScript (dinamika manipulasi elemen) pada Internal di dalam tag script, pada Eksternal file terpisah .js dihubungkan dengan via script src, atau pada Event-Driven dengan onclick atau input pengguna.

C. Keuntungan Menggunakan CSS dan JavaScript

Keuntungan CSS

  • Konsistensi dan Estetika (membuat tampilan website seragam di semua perangkat dan browser, mudah diperbarui, serta menambah daya tarik dengan desain yang keren).
  • Responsivitas (memastikan website tampil bagus di berbagai ukuran layar, seperti ponsel atau laptop, dengan bantuan media queries).
  • Efisiensi (mengurangi kode berulang menggunakan kelas dan selektor, memisahkan desain dari HTML, sehingga mengelola gaya jadi lebih gampang).

Keuntungan JavaScript

  • Interaktivitas dan Dinamis (membuat website lebih hidup dengan fitur seperti dropdown menu atau validasi formulir, dan bisa memperbarui konten tanpa perlu memuat ulang halaman menggunakan teknik seperti AJAX).
  • Fungsionalitas dan Fleksibilitas (mendukung pembuatan aplikasi web canggih, bisa digunakan di sisi klien maupun server (Node.js), serta cocok untuk berbagai browser dan perangkat).
  • Efisiensi dan Ekosistem Luas (menghemat waktu dengan mengurangi kebutuhan reload halaman, ditambah banyak library dan framework seperti React atau Vue yang mempermudah pengembangan).

D. Keuntungan Menggunakan Bootstrap

  • Komponen siap pakai (karena ada banyak elemen keren seperti tombol, navbar, formulir, atau card yang tinggal pakai, jadi hemat waktu buat desain).
  • Efisien waktu (karena kombinasi class CSS dan JavaScript sederhana bikin coding lebih cepat tanpa mulai dari nol).
  • Tampilan responsif (karena website langsung nyaman dilihat di semua perangkat, dari HP sampai laptop, tanpa ribet).
  • Dokumentasi dan komunitas besar (karena dokumentasinya jelas, banyak tutorial, dan komunitasnya besar, jadi gampang cari bantuan).
  • Mudah dikustomisasi (karena gampang diubah sesuai kebutuhan pakai variabel Sass atau tema, cocok buat proyek apa saja).
  • Tampilan konsisten (karena semua bagian website kelihatan rapi dan seragam, membuat situs terlihat pro dan premium).
